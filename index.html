    Corendog

    Voor dit project moesten we een website maken om een doelgroep te helpen in de Corona crisis. Dit project heb ik samen met een klasgenoot gemaakt.

    Het idee achter de website is dat mensen die in een risicogroep een account kunnen aanmaken op onze site om vervolgens in contact te kunnen komen met hondenuitlaat vrijwilligers. Via onze site kun je jezelf ook opgeven als vrijwilliger.

    Van deze site heb ik de frontend geprogrammeerd en de klasgenoot de backend. Het project is gebouwd in MVC model en de gebruikte programmeertalen zijn HTML, CSS, PHP en JavaScript.
